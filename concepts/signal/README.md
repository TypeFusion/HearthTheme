# Signal Concept Pack

Updated: 2026-04-05

`SignalCode` is a working concept for the second public-facing flavor after `HearthCode`.

This folder is intentionally outside `color-system/schemes/`.
It is a safe staging area for product and palette work before anything enters the generated pipeline.

## Why This Exists

The repo already proved that multiple flavors can coexist technically.

What it has not yet proved is a second flavor with enough product distance to matter.

`SignalCode` is the first serious attempt at that difference:

- more contrast-forward
- more vivid at semantic landmarks
- more energetic in active coding
- still disciplined enough for everyday use

## Files

- `scheme.json`: a scheme-manifest draft for the concept
- `philosophy.md`: the flavor's human-readable design thesis
- `taxonomy.json`: abstract grouping of families, roles, and interaction lanes
- `foundation.json`: top-level family palette draft across four climates
- `semantic-rules.json`: role-to-family mapping for syntax semantics
- `surface-rules.json`: abstract surface derivation draft
- `interface-rules.json`: shell and navigation hierarchy draft
- `interaction-rules.json`: cursor, selection, focus, and status draft
- `guidance-rules.json`: brackets, guides, and whitespace draft
- `feedback-rules.json`: note/info/success/warning/error draft
- `terminal-rules.json`: terminal semantic palette draft
- `variant-knobs.json`: climate-tuning draft
- `role-energy-map.json`: role priorities and semantic energy targets
- `dark-flagship-palette.json`: a first dark-only family and role direction
- `signal-preview.svg`: a dark flagship visual mockup for quick design review

## Current Design Position

This concept is intentionally not:

- a cooler `Hearth`
- an olive-led atmosphere
- a stone-neutral sibling
- a neon spectacle

It is a yellow-green-led, contrast-forward, high-recognition direction with:

- carbon substrate
- citrus/chartreuse signal family
- electric blue callable anchor
- coral interruption pressure

The design target is "faster structure recognition" rather than "more visual drama".
