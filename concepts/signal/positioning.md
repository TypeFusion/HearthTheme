# Signal Positioning

## One-Line Position

SignalCode is the sharper, more vivid companion to HearthCode for users who want faster landmarks and a more energized coding mood.

## Fast Contrast

### HearthCode

- warm-neutral
- low-glare
- steady
- long-session reading friendly

### SignalCode

- contrast-forward
- high-recognition
- vivid at key roles
- active-coding friendly

## When Someone Should Pick Signal

Signal is the better fit when a user wants:

- faster scanning across dense files
- clearer syntax landmarks
- stronger separation between keyword, function, and type
- a theme that feels more awake than calm

## When Someone Should Still Pick Hearth

Hearth is still the better fit when a user wants:

- quieter long-session reading
- warm-neutral atmosphere
- less visual jump between syntax roles
- a more settled editing mood

## Internal Positioning Rule

Signal should always be explainable as a different working posture, not just a different palette.
